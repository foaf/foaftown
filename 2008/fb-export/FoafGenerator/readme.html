About:

The Facebook Foaf generator is part of a wider research aim to integrate social identities from web 2.0 style social networking sites. The main problem with information integration from various different sources is the issue of different data formats. We cannot simply compare raw data in xml with data in a tabular text format, there must be some mediator format that allows easier integration and compuations to be performed, hence the generation of information in Foaf format.


The tool works by extracting information from Facebook and then compiling the information into the correct format according to the Foaf specification. The file can then be downloaded and merged with other foaf files a user may have.


As social networks are careful not to release certain sensitive information to third party applications, this inhibits the email address of any users from being extracted. This can cause problems when conforming to the Foaf specification as the mbox is required as a unique identifier for the person in question. To alleviate this problem, a new ontology has been created to represent social networks. Each social network account has a unique identifier, in the case of facebook this is the userid.
